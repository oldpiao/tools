技术点:
+ 分词前处理
    + 文本分句
    + 文本分子句
+ 自定义词类  # MyPair
+ 分词
    + 带词性分词
    + 正则分词  # 原正则取词，修改后作为第二类分词模块使用，可以在分词前优先使用正则取词，再对剩余部分分词，涉及逻辑会在其他位置描述
+ 分词后处理
    + 停用词库
    + 正则停用词库
    + 自定义词库
    + 特殊符号  # 《aaaaaa》、“bbbbb”
    + 同义词处理
        + 同义词库
        + 同义正则
    + 低频词处理
    + 词性正则  # 创造一种类似正则的文本处理格式，可以识别词性，并根据词性、词义优化分词

+ 添加各种词典
    + 多种类词典
        + 可以设置多种词典，针对不同的问题使用不同的词典
    + 多级配置
        + 目前思路是三级词典,根据优先级: 用户自定义词典>分类词典>默认词典
        + 其中默认词典可能不需要，根据具体情况定
    + 支持热配置
+ 建议分词逻辑处理优先级
    + 特殊符号内内容>用户自定义词典>停用词词典>自定义分词算法>分类词典>默认词典>算法分词
+ 建议处理流程
    + 文章分句-->文章分词-->新词组合-->同义词-->停用词处理-->低频词
